Проект для реализации SCADA подобной системы.
 Будет иметь модель с указанием опрашиваемых регистров, как в онлайн режиме так и в архивную базу.
 необходимо реализовать Modbus TCP и RTU.
 В качестве подопотного клиетна использовать контроллер owen - ethernet,
    rs-485- эмитировать через rs-232-usb(проверить связку)
